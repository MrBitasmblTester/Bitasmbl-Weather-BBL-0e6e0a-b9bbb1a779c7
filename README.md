# Bitasmbl-Weather-BBL-0e6e0a-b9bbb1a779c7

## Description
Build a web application that displays current weather, multi-day forecasts, and visualizes trends with charts. The project focuses on integrating APIs, dynamic front-end updates, and data visualization for an interactive user experience.

## Tech Stack
- FastAPI
- ASP.NET Core
- React

## Requirements
- Fetch real-time weather data from a public API
- Display multi-day weather forecasts with clear UI elements
- Visualize temperature, humidity, or other metrics with charts
- Support user interactions, such as searching by city
- Optionally cache API responses to improve performance

## Installation
bash
git clone https://github.com/MrBitasmblTester/Bitasmbl-Weather-BBL-0e6e0a-b9bbb1a779c7.git

## Usage
- Run chosen backend (FastAPI or ASP.NET Core) and React frontend according to your environment.

## Implementation Steps
1. Set up FastAPI or ASP.NET Core backend project.
2. Integrate a public weather API client.
3. Expose endpoints for current weather and multi-day forecasts.
4. Add optional caching for API responses.
5. Scaffold React app for UI.
6. Implement city search and display current/forecast data.
7. Add chart components for selected metrics.
8. Wire charts to backend data and refine interactions.

## API Endpoints
- GET /weather/current
- GET /weather/forecast
